XLSegment 是一个目前有三种 Style 的 Segment, 支持滚动.
写这个是因为在[TKDotSegment](https://github.com/TBXark/TKDotSegment)的时候,发现不支持 scroll 的滚动, 就产生了自己再写一个的想法.
